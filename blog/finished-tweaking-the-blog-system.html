A couple days ago, I started using the 'lb' script by Luke Smith. It is basically a simple blog system. So I started changing stuff and now it's done. What can I say? Surely, reading and changing shell scripts is dead easy. Just learn it, man. It's good for you. The hard part, for me, is understanding commands like sed, awk, and the like, relgular expression as well. Mind you, I have no experience learning any programming language or scripting at all. Although I'm quite used to using command lines, cli programs and configuring them a long time ago, I've never really used sed or stuff like that. So, the main change I made to the 'lb' script, is adding the two line of code, which is not put in the script itself, shown in one of his video, the function that write the latest few blog entries into the home page. Anyway, thank you, Luke, for making this script. I do hope he will add his tagging script into the blogging system (He did post the commands doing it on his site, but it's still rocket science to me.), though. That way, it would become perfect. This is Free *nix magic, bro.
